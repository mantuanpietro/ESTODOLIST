# Requisitos funcionais - Gerenciador de Tarefas
1. RF01 - O sistemas deve permitir a adição de uma tarefa.
2. RF02 - O sistema deve listar as tarefas existentes.
3. RF03 - O sistemas deve permitir a edição de uma tarefa existente.
4. RF04 - O sistema deve permitir a exclusão de uma tarefa.
5. RF05 - O sistema deve permitir marcar uma tarefa como concluida.
6. RF06 - O sistema deve permitir a pesquisa de tarefas.
7. RF07 - O sistema deve permitir a filtragem de tarefas.
